# Crafty fonts
Familiar pixel fonts for any Luanti game thanks to macsMinecraft by macimas and Monocraft by Idrees.

## Font variants
- Regular
- Bold
- Italic
- Bold italic
- Mono-spaced
- Mono-spaced bold
- Mono-spaced italic
- Mono-spaced bold italic

## How to use
Install and enable like any other mod.
